3. 以下是解决进程互斥进入临界区的一种解法。

P:

	```C
	pturn = true;
	while (qturn) ;
		临界区操作
	pturn = false;
	```

Q:

	```C
	qturn = true;
	while (pturn) ;
		临界区操作
	qturn = false;
	```

其中，pturn、qturn的初值为false

如果P、Q两个进程同时想进入临界区，那么会发生下面哪一种情形？

* **P和Q都进入不了临界区**
* P和Q都进入了临界区
* P先进入临界区，Q再进入临界区
* Q先进入临界区，P再进入临界区
